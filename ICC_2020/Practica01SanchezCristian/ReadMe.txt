No fueron problemas muy complicados, y el único problema que tuve fue recordar
que el tipo de dato primitivo long, float, double, necesitan un carácter especifico para
indicar que se está usando ese tipo de dato.

Para compilar la práctica, solo basta con abrir la terminal, dirigirse a la carpeta donde se
encuentra el archivo .java, es decir Practica1.java, y escribir en la terminal
$ javac Practica1.java
Para generar el archivo .class, y para correr la practica escribimos:
$ java Practica1

Sería todo.
